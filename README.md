# 📝 ToDo Manager (C# Console Application)

A simple console-based ToDo Manager written in C#.

This project allows users to create and manage tasks using a menu-driven interface.

## Features

- ➕ Add new tasks
- 📋 Show all tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🔍 Search tasks by name
- 📌 Show completed tasks
- 📌 Show uncompleted tasks
- ✏️ Edit task names

## Technologies

- C#
- .NET
- Console Application
- List<T>
- Methods
- Loops
- Conditional Statements
